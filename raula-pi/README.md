The raula-pi is a python module to collect data from sensors and camera. It is not specific for this project, you can use it anywhere you need a generic data collector for raspberry pi.

# Current modules

Sensors:
* SenseHat
* IBS-TH1 thermometer
* GrovePi [WIP]

Cameras:
* Raspi Camera

Cloud:
* ThingsBoard
* AWS S3
